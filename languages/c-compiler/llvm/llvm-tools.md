# LLVM Tools

**From:** [LLVM](../../../index.md#clang-specific)

## Tools

### Emit llvm

```
clang -O3 -emit-llvm test.cc -c -o hello.bc
```

### Run llvm jit

```
lli hello.bc
```

### Explore the IR

```
llvm-dis < hello.bc | less
```

### Compile IR into ASM

```
llc hello.bc -o hello.s
```
