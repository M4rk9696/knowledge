# Job file table

[Back](../../index.md){: .button}

JFT is a DOS data structure in the Program Structure Prefix

Handle  |   Description
-----------------------
0       |   Stdin
1       |   Stdout
2       |   Stderr
3       |   COM1
4       |   LPT1

### Example usage

```
int 21h
```
