# GNU Plot Terminal

[Back](../index.md#gnuplot)

A GNU plot terminal is the output for gnuplot

http://gnuplot.info/docs_5.5/Terminals.html

```
# pngcairo terminal
set terminal pngcairo size 350,262 enhanced font 'Verdana,10'
set output 'out.png'

set term pdfcairo font "Arial,12"
set output
set output 'out.pdf'

```
